# Day 002 - Deploy an Azure Virtual Machine

## 🎯 Challenge

Create an Azure Virtual Machine that meets the following requirements:

- Existing Resource Group
- VM Name: **nautilus-vm**
- Region: **East US**
- Ubuntu Server 24.04 LTS
- VM Size: **Standard_B1s**
- Allow inbound SSH (Port 22)
- 30 GB Standard HDD OS Disk
- Default configuration for all remaining settings
- Verify SSH connectivity

---

## 🛠️ Azure Services Used

- Azure Virtual Machine
- Resource Group
- Virtual Network
- Network Security Group
- Public IP
- Azure CLI

---

## 📋 Implementation Steps

### Step 1
Retrieve Azure credentials.

### Step 2
Login to Azure CLI.

### Step 3
Deploy the Virtual Machine.

### Step 4
Verify networking configuration.

### Step 5
Connect using SSH.
